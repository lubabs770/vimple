Lesson seven: macros ( {record_macro}q ... {record_macro} , then {play_macro}q ).

Add a one-line doc comment immediately above each of the four exported
functions (addTask, listOpen, completeTask, removeTask) so each reads:
    /** todo op */
    export function ...

How: position above addTask. Record a macro: {record_macro}q  then
{open_above}  type  /** todo op */  Esc, move down to the next function,
{record_macro} to stop. Replay with {play_macro}q on the others. Save with :w
